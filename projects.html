프로젝트 경험

제가 직접 참여하고 구현했던 주요 프로젝트들을 소개합니다. 각 프로젝트에서 사용된 기술과 제가 담당한 역할을 상세하게 기록했습니다.

소담일기 - 시각장애인을 위한 감성 이미지 캡셔닝 서비스

주요 내용: 시각장애인 사용자를 위해 이미지와 음성 입력을 받아 경량화된 BLIP/CLIP 모델로 이미지 캡셔닝을 처리하고, LLM API를 활용해 감성적이고 개인화된 설명을 제공하는 서비스입니다.

  • 기술 스택: Python, Django, BLIP, CLIP, LLM API
  • 담당 역할: Django를 이용한 백엔드 아키텍처 설계, 모델과 앱의 안정적 연동을 위한 모듈화, 초기 테스트 및 로직 검증

AI 모델 API 연동 프로젝트

주요 내용: BLIP 모델을 다운로드하여 API 서버에 연동하고, Postman을 이용해 연결성을 검증했습니다.

  • 기술 스택: Python, API, Postman, BLIP 모델
  • 담당 역할: 모델 로드 및 서버 연동, API 엔드포인트 테스트, 결과 검증

Intel AI Workforce 기반 배포 프로젝트

주요 내용: OpenVINO 툴킷을 활용하여 AI 모델을 다양한 하드웨어 환경(CPU, GPU, NPU)에 최적화하여 배포하는 프로젝트를 진행했습니다.

  • 기술 스택: OpenVINO, Python, 이종 실행(Heterogeneous Execution)
  • 담당 역할: 모델 IR 변환, 멀티 디바이스 추론 로직 구현, 성능 최적화

안드로이드 앱 개발 (Kotlin & Compose)

주요 내용: 안드로이드 스튜디오를 이용해 Kotlin과 Jetpack Compose로 모바일 앱을 개발했습니다. 비동기 처리와 스레드 분리를 통해 UI/UX를 개선하는 로직을 구현했습니다.

  • 기술 스택: Kotlin, Android Studio, Jetpack Compose, 비동기 처리
  • 담당 역할: 앱 UI/UX 설계, 백그라운드 작업 구현, 모듈화된 코드 작성

Unity 기반 3D 가상환경 설계

주요 내용: Unity 엔진과 C#을 사용하여 간단한 3D 상호작용 가상환경을 설계했습니다. 객체 간의 상호작용 및 물리 법칙을 구현했습니다.

  • 기술 스택: Unity, C#, 3D 가상환경
  • 담당 역할: 가상환경 구성, 상호작용 스크립트 작성, 물리 엔진 설정
